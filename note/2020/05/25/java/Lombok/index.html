Lombok

2020-05-25
  • java

提供了一组注解，用于消除java类中的大量样板代码，解放生产力

引入lombok

<dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <version>...</version>
</dependency>

在IDEA中使用

安装插件市场提供的lombok插件即可

@Builder

  • 源代码
package com.ccbft.antigoods.stock;

import lombok.Builder;

@Builder
public class Person {
    // 姓名
    private String name;
    // 性别
    private String sex;
    // 年纪
    private Integer age;
    // 工资
    private Integer salary;

    public static void main(String[] args) {
        Person person = Person.builder().name("yanhui").sex("M").age(19).build();
        System.out.println(person);
    }
}
