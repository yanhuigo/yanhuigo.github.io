接入微信，成为开发者

成功注册登录测试平台，获取appID和appsecret

  1. 填写服务器地址，必须在公网80端口部署服务；微信会像指定的地址发送一个GET请求，服务器验证成功后即可成为开发者
  2. 填写token，与服务端对应，服务端会把他加入消息校验的因子
  3. 接入后微信用户给公众号发消息和点击公众号相关的菜单都会给接入的服务器发一个POST消息请求

接入JAVA实现

@GetMapping(produces = "text/plain;charset=utf-8")
public String authGet(@PathVariable String appid,
                      @RequestParam(name = "signature", required = false) String signature,
                      @RequestParam(name = "timestamp", required = false) String timestamp,
                      @RequestParam(name = "nonce", required = false) String nonce,
                      @RequestParam(name = "echostr", required = false) String echostr) {

    log.info("\n接收到来自微信服务器的认证消息：[{}, {}, {}, {}]", signature, timestamp, nonce, echostr);

    if (StringUtils.isAnyBlank(signature, timestamp, nonce, echostr)) {
        throw new IllegalArgumentException("请求参数非法，请核实!");
    }

    if (checkSignature(timestamp, nonce, signature)) {
        return echostr;
    }

    return "非法请求";

}

private boolean checkSignature(String timestamp, String nonce, String signature) {
    try {
        String token = "926484";
        return gen(token, timestamp, nonce).equals(signature);
    } catch (Exception e) {
        log.error("Checking signature failed, and the reason is :" + e.getMessage());
        return false;
    }
}


private static String gen(String... arr) {
    if (StringUtils.isAnyEmpty(arr)) {
        throw new IllegalArgumentException("非法请求参数，有部分参数为空 : " + Arrays.toString(arr));
    }

    Arrays.sort(arr);
    StringBuilder sb = new StringBuilder();
    for (String a : arr) {
        sb.append(a);
    }
    return DigestUtils.sha1Hex(sb.toString());
}

发送模板消息JAVA实现

public static final String API_DEFAULT_HOST_URL = "https://api.weixin.qq.com";
public static final String MESSAGE_TPL_SEND = API_DEFAULT_HOST_URL + "/cgi-bin/message/template/send";

public void tplMessage(String message, String accessToken) {

        CloseableHttpClient httpClient = HttpClientBuilder.create().build();

        String uriWithAccessToken = MESSAGE_TPL_SEND + "?" + "access_token=" + accessToken;
        HttpPost httpHost = new HttpPost(uriWithAccessToken);

        StringEntity stringEntity = new StringEntity(String.format("{\"touser\": \"oCqsh6D_XB1jmC3s4f-Eai4uzQ98\",\"template_id\": \"ruDmCTrIIwg2JpXQ5dMK0h2PmKobpUboKsS5hiAgd-E\",\"data\": {\"random\": {\"value\": \"%s\"}}}", message), Consts.UTF_8);
        stringEntity.setContentEncoding("UTF-8");
        stringEntity.setContentType("application/json");
        httpHost.setEntity(stringEntity);

        CloseableHttpResponse response = null;
        try {
            response = httpClient.execute(httpHost);
            log.info("【微信请求响应】\n" + EntityUtils.toString(response.getEntity()));
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            try {
                httpClient.close();
            } catch (IOException e) {
                log.error("关闭httpClient异常", e);
            }

            if (response != null) {
                try {
                    response.close();
                } catch (IOException e) {
                    log.error("关闭CloseableHttpResponse异常", e);
                }
            }
        }

}
